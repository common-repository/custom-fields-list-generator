Create a custom fields list from field labels on the posts of any type!

== Description ==

Scroll to a target position and highlight it if you click on the list item.


=== features ===

* Items name is got from custom field title.
* You can create a list of custom fields up to the first level.
* You can hide a list for each post type.
* And you can hide it for each post if you input post id separated by comma in textarea.
* If you sort or show / hide fields on a post, the list will be updated when you reload a browser.


=== How to use ===

1. Activate this plugin.
2. Check target posts and click on “Save”.
3. An icon is shown your post.

  1. This is shown next to the public widget in the upper right if you use a Gutenberg Editor.
  2. This is pinned to the screen at the bottom right if you use a Classic Editor.

4. A custom field list is shown if you click the icon.
5. Scroll to a target position if you click on the list item.
6. Scroll to a top of page if you click “Back to Top”.


== Installation ==

1. From the WP admin panel, click “Plugins” -> “Add new”.
2. In the browser input box, type “Custom Fields List Generator”.
3. Select the “Custom Fields List Generator” plugin and click “Install”.
4. Activate the plugin.

OR…

1. Download the plugin from this page.
2. Save the .zip file to a location on your computer.
3. Open the WP admin panel, and click “Plugins” -> “Add new”.
4. Click “upload”.. then browse to the .zip file downloaded from this page.
5. Click “Install”.. and then “Activate plugin”.
